# Laravel CRUD Example

## Development
1. Download project
```bash
$ git clone https://github.com/mnindrazaka/laravel-crud-example
```

2. install library
```bash
$ composer install
```

3. rename .env-example to .env

4. Set database configuration on .env

5. Migrate database
```bash
$ php artisan migrate
```

6. Seed database
```bash
$ php artisan db:seed
```

7. Run server
```bash
$ php artisan serve
```
